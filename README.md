# Week 0: Developer Workshop Check

Welcome to your first assignment! The goal here is to prove your Git and IntelliJ pipeline is working.

## Your Task:
1. Open `src/Main.java` in IntelliJ IDEA.
2. Find the `getStatus()` method.
3. Change the text from `"NOT READY"` to exactly `"SETUP COMPLETE"`.
4. Run the code locally using the green play button to ensure it prints `SETUP COMPLETE`.
5. Commit and push your changes to GitHub.
6. Check GitHub to see if you got the green ✅!
